# お掃除 TODO アプリ API 設計書

## 概要

本 API は、お掃除 TODO アプリのフロントエンド（Next.js）と通信し、お掃除タスク（tasks）データの CRUD（登録・取得・更新・削除）
、担当者（users）、カレンダー表示用データを提供します。

### 🔐 認証・権限（Authorization）

本アプリでは、ユーザーのログイン状態に応じて利用可能な API を制限する。

| 状態                     | 内容                                             |
| ------------------------ | ------------------------------------------------ |
| **ゲスト（未ログイン）** | カレンダーの閲覧のみ可能                         |
| **会員（ログイン済み）** | タスクの登録・編集・削除が可能（自分の操作のみ） |

| リソース         | 説明                                                |
| ---------------- | --------------------------------------------------- |
| `/api/tasks`     | お掃除タスク（1 件の掃除）の CRUD 管理              |
| `/api/cleanings` | 掃除箇所（トイレ / お風呂 / コンロ など）の一覧取得 |
| `/api/users`     | 担当者一覧（A さん / B さんなど）の取得             |
| `/api/status`    | お掃除状態（完了・未完了）の取得                    |
| `/api/calendar`  | カレンダー表示用のタスク集計データ取得              |

### 各エンドポイントの認可ルール

| HTTP メソッド | エンドポイント                | 機能概要                         |
| ------------- | ----------------------------- | -------------------------------- |
| GET           | `/api/tasks`                  | 全タスク一覧を取得               |
| GET           | `/api/tasks/:id`              | 指定 ID のタスクを取得           |
| POST          | `/api/tasks`                  | 新しいタスクを追加               |
| PUT           | `/api/tasks/:id`              | 既存のタスクを更新               |
| DELETE        | `/api/tasks/:id`              | 既存のタスクを削除               |
| GET           | `/api/cleanings`              | 掃除箇所の一覧を取得             |
| GET           | `/api/users`                  | 担当者一覧を取得                 |
| GET           | `/api/status`                 | お掃除状態一覧を取得             |
| GET           | `/api/calendar?month=2026-01` | その月のカレンダーのタスクを取得 |

## データ形式

### 🧹 タスクデータ（Task）

| フィールド名 | 型     | 説明                                       |
| ------------ | ------ | ------------------------------------------ |
| id           | number | 自動採番 ID                                |
| date         | string | 日付（例：`"2026-01-11"`）                 |
| time         | string | 時間（例：`"10:00"`）                      |
| cleaning_id  | number | 掃除箇所 ID（例：1 → 「トイレ」）          |
| user_id      | number | 担当者 ID（例：1 → A さん）                |
| status       | string | `"done"`（完了） or `"not_done"`（未完了） |
| memo         | string | 任意メモ（省略可）                         |

### 🏠 掃除箇所データ（cleanings）

| フィールド名 | 型     | 説明                                                 |
| ------------ | ------ | ---------------------------------------------------- |
| id           | number | 自動採番 ID                                          |
| name         | string | 掃除箇所名（例：「トイレ」「お風呂」「コンロ」など） |

### 👤 担当者データ（User）

| フィールド名 | 型     | 説明                                 |
| ------------ | ------ | ------------------------------------ |
| id           | number | 自動採番 ID                          |
| name         | string | 担当者名（例：「A さん」「B さん」） |

### 📅 カレンダー API レスポンス形式例

{
"month": "2026-01",
"tasks": [
{
"date": "2026-01-11",

}
